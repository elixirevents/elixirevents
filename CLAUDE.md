# ElixirEvents

Open source community directory for Elixir & BEAM conferences, talks, speakers, and topics. Domain: elixirevents.org

## Design Context

### Users
Elixir/BEAM developers at all levels. They discover events, rewatch talks, find speakers, and browse topics. Contributors submit data via YAML + pull requests.

### Brand Personality
**Playful, Warm, Community-driven.** The "Elixir" = magic potion. Potion bottle logo, sparkle decorations, purple brand color `#8a3897`. Fun is a feature, not decoration.

### Aesthetic Direction
- **Dark theme is the guiding star** — design dark-first, then adapt for light
- Warm editorial meets magical playfulness
- Bricolage Grotesque (display) + DM Sans (body)
- Deep plum dark base, warm cream light base — never pure black/white
- Secondary: warm coral. Accent: teal/sage
- Anti-references: cold corporate, "AI slop" aesthetics, dated directory designs

### Design Principles
1. **Events are the star** — don't let chrome overshadow content
2. **Inclusive by default** — no "featured" pedestals, let data speak
3. **Dark-first, both-excellent** — dark is primary, light is companion
4. **Fun is a feature** — potion motif and sparkles are the brand
5. **Community over curation** — open source ethos in every design decision

### Accessibility
WCAG AAA where practical. Enhanced contrast, `prefers-reduced-motion` support, semantic HTML, keyboard navigation.

See `.impeccable.md` for the full design specification.
